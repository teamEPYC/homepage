## About the Role

We're looking for a Senior Frontend Developer to join our engineering team and help build the next generation of privacy-focused blockchain applications. You'll work on cutting-edge technologies including zero-knowledge proofs, blockchain infrastructure, and modern web applications.

## What You'll Do

- Build and maintain user-facing applications using React, TypeScript, and modern web technologies
- Collaborate with our design team to create intuitive and beautiful user interfaces
- Work closely with backend engineers to integrate blockchain functionality
- Optimize applications for performance and scalability
- Contribute to our design system and component libraries
- Mentor junior developers and participate in code reviews
- Stay up-to-date with the latest frontend technologies and best practices

## What We're Looking For

### Required Skills

- 5+ years of experience with modern JavaScript/TypeScript
- Strong proficiency with React and its ecosystem
- Experience with state management libraries (Redux, Zustand, etc.)
- Solid understanding of HTML, CSS, and responsive design
- Experience with build tools (Vite, Webpack, etc.)
- Knowledge of testing frameworks (Jest, React Testing Library)
- Understanding of Git and collaborative development workflows

### Nice to Have

- Experience with blockchain technologies or Web3 applications
- Knowledge of Next.js or React Router
- Experience with design systems and component libraries
- Understanding of cryptography concepts
- Experience with performance optimization and monitoring
- Familiarity with accessibility standards (WCAG)

## About Miden

Miden is building the privacy-focused execution layer for the modular blockchain stack. We're a small but growing team of engineers, researchers, and designers who are passionate about creating technology that preserves user privacy while enabling new forms of digital interaction.

## Application Process

1. **Initial Review** - We'll review your application and get back to you within 3 business days
2. **Technical Screen** - 45-minute technical discussion about your experience and approach
3. **Take-Home Project** - Build a small application using our tech stack (2-3 hours)
4. **Final Interview** - Meet with team members and discuss the take-home project
5. **Reference Check** - We'll connect with your references
6. **Offer** - If everything goes well, we'll extend an offer!

The entire process typically takes 1-2 weeks from application to offer.
